# Visualist Agent — QShift Variant (Simple Charts)

Extends `visualist.md`. Read that file first — this document lists only the
differences that apply to QShift content.

---

## Chart type restriction

QShift charts must be immediately readable at a glance without interpretation.
**Only bar and line charts.** No Sankeys, no geographic scatter maps, no
demographic pyramids, no dual-axis charts.

If a placeholder description implies a chart type not on this list,
produce the closest readable bar or line equivalent and note the simplification
in the caption.

Maximum **two charts per piece**. One is often better.

## Chart titles

QShift chart titles state a practical finding, not an analytical one:
- WaywardHouse: "Who Arrived: Age Profile of In-migrants vs Alberta Population"
- QShift equivalent: "New arrivals skew 20 years younger than the existing workforce"

The QShift reader should understand the chart's point before looking at the data.

## Hero image

Same tiers as `visualist.md`, but tone brief changes:
- Tone: `professional / strategic / authoritative`
- Avoid: anything that reads as academic or data-heavy; abstract textures preferred
  over data visualisations as hero background
- Typography should dominate — title large and legible on a clean background
